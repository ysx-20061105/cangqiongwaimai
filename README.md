## 项目介绍
### 业务流程
整个点餐系统的业务流程：

1. 管理员可以管理员工、新增菜品、管理套餐等
2. 用户登陆后，可以使用小程序进行点餐、加入购物车、下单、支付、催单
3. 管理员可以接受来单提醒和用户的催单
4. 管理员可以在后台统计分析、导出营收数据
